# Mega6502 [![Badge License]][License]

*An experimental implementation of a minimal* <br>
*8 - bit platform using an **Arduino** + **6502** chip.*

<br>

## Hardware

The **Mega6502** requires:

 **1 x** `Ardunio Mega 2560` development board 
  
 **1 x** `CMOS 65c02`

<br>
